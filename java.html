JavaScript

JavaScript gjør nettsiden din mer levende! Når du klikker, skriver inn tekst eller beveger musa, er det JavaScript som styrer det.

Hva kan JavaScript gjøre?

JavaScript kan reagere på brukerhandlinger som klikk, skrivefelt, og endre stil på elementer. Det kan også lagre verdier og vise meldinger.

Eksempel: Endre farge på boksen

Klikk på boksen for å endre farge. Dette viser hvordan JavaScript kan gjøre siden interaktiv.

Eksempel 2: Si hei med navn

Skriv navnet ditt i boksen under og få en personlig hilsen når du klikker på knappen.
